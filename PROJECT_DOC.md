# Документация проекта: Дашборд администратора коворкинга

## 1. Что это?
**AdminFlow** — это единая панель управления (дашборд) для администратора интернет-кафе и коворкинга. Приложение агрегирует входящие запросы от клиентов из различных каналов (мессенджеры, социальные сети, электронная почта) в единый структурированный список задач.

## 2. Для кого?
*   **Администраторы коворкингов:** для эффективной обработки потока задач и соблюдения дедлайнов.
*   **Владельцы бизнеса:** для контроля качества обслуживания, мониторинга нагрузки и анализа популярных услуг.
*   **Клиенты (косвенно):** получают более быстрый и качественный сервис без потери их сообщений.

## 3. Какую проблему решает?
*   **Хаос в коммуникациях:** сообщения приходят в WhatsApp, Telegram, VK, почту — администратор разрывается между вкладками.
*   **Пропуск заявок:** из-за большого потока (до 100 человек в день) легко забыть про сообщение в мессенджере.
*   **Нарушение сроков:** отсутствие наглядного трекинга дедлайнов для сложных задач (редактирование документов, дизайн).
*   **Сложность работы с файлами:** необходимость постоянно скачивать и пересылать файлы из разных источников.

## 4. Основные функции (MVP)
*   **Единая таблица задач:** отображение всех входящих заявок с указанием источника (VK, Email, Telegram и т.д.).
*   **Статусы задач:** "Новая", "В работе", "Ожидает оплаты", "Готово", "Выдано".
*   **Управление файлами:** просмотр прикрепленных документов, фотографий и прослушивание аудиосообщений прямо в интерфейсе.
*   **Карточка клиента:** история обращений, контактные данные и уровень цифровой грамотности (для подбора стиля общения).
*   **Фильтрация и поиск:** быстрый поиск по типу услуги (печать, фото, ремонт) или имени клиента.
*   **Уведомления о дедлайнах:** визуальная индикация задач, срок выполнения которых подходит к концу.

## 5. Чего НЕ будет на старте
*   **Автоматические ответы (AI-боты):** на первом этапе общение остается за человеком.
*   **Интеграция с кассой:** учет оплат ведется вручную или во внешней системе.
*   **Автоматическое редактирование документов:** программа только хранит файлы, редактирование происходит в Word/Excel.
*   **Мобильное приложение для клиентов:** взаимодействие идет через существующие мессенджеры.

## 6. Технологический стек
*   **Frontend:** React + TypeScript + Tailwind CSS.
*   **Анимации:** Motion (framer-motion).
*   **Иконки:** Lucide React.
*   **Backend (планируется):** Node.js/Express для интеграции с API мессенджеров.
